Why Cockpit?

Self-hosted

Keep complete control. Your data is and will be always yours.

Open Source

We Open Source and Cockpit is our contribution we would like to share with the world.

Clients will love it

Cockpit comes with an intuitive design and UX which makes it a joy to manage the content.

Cockpit is awesome if you need a flexible content structure but don't want to be limited in how to use the content. Cockpit is a perfect match if you want to support multiple devices or need a content management UI for static site builders. Build unique applications and let Cockpit feed them with content.

Cross-Channel Content Distribution

Web, App, Smart Device, IoT

Some key features

Simple. Powerful. Open Source. Enjoy developing.

  • Dead simple installation in less than one minute. Kickstart your your next project without any hassle.

  • A simple but powerful headless CMS that is built around an API-first and JSON content approach.

  • SQLite or MongoDB, your choice. Use Cockpit for quick and simple projects or utilize MongoDB for larger ones.

  • A system which invites to extend with custom code. Build a solution to any problem.

  • Cockpit is not a web site builder. It is a content provider. Consume the content the way you want.
